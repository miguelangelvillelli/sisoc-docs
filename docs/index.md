Esta documentación define el **MVP** del módulo **Legajo de Espacio** para que cada espacio comunitario pueda acceder desde una **app móvil** a información útil de su relación con el Programa.

## Qué buscamos
- Valor visible temprano en móvil (uso real en territorio)
- Trazabilidad operativa para gestión y auditoría
- Base sólida para evolucionar a “organización” y funcionalidades ampliadas
