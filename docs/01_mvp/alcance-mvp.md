# Alcance MVP

!!! warning "Regla"
    Todo lo que no esté definido acá entra como **Release 2**. No se suman excepciones sin ADR.

## Objetivo
Que cada espacio comunitario tenga acceso desde una app móvil a información útil sobre su relación con el Programa.

## Incluye (MVP)
- Información institucional y convenios
- Nóminas
- Prestación alimentaria (definir si es solo visualización o carga mínima)
- Formación
- Rendiciones y comprobantes

## No incluye (por ahora)
- Funcionalidades avanzadas de capacitación
- Paneles extendidos / analítica avanzada
- Automatizaciones complejas (más allá de notificaciones básicas si aplican)

## Criterio de éxito
El referente del espacio puede completar tareas clave desde el teléfono sin asistencia técnica.
