# ADR-0002 · Usuario organización

**Fecha:** 2026-02-03  
**Estado:** Propuesto  
**Decisor/es:** Dirección + Equipo SISOC  

## Contexto
Surge la necesidad de contemplar organizaciones que administran más de un espacio, con vista consolidada.

## Decisión
Incorporar el concepto “Organización” y un rol/usuario asociado que pueda visualizar múltiples espacios (y, eventualmente, operar).

## Consecuencias
- Impacta UX (selector de espacio / dashboard consolidado).
- Impacta RBAC y auditoría.
- Puede implementarse por etapas: primero solo visualización.
