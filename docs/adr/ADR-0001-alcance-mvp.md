# ADR-0001 · Alcance MVP

**Fecha:** 2026-02-03  
**Estado:** Propuesto  
**Decisor/es:** Dirección + Equipo SISOC  

## Contexto
Necesitamos un MVP operativo para espacios comunitarios con foco mobile-first.

## Decisión
El MVP incluye cinco módulos: información institucional, nóminas, prestación alimentaria, formación, rendiciones.

## Consecuencias
Se posterga a Release 2 todo lo no esencial (capacitaciones avanzadas, paneles extendidos, etc.).
