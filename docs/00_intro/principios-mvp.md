# Principios del MVP

!!! info "Estado"
    **Versión:** v0.1  
    **Última actualización:** 2026-02-03  
    **Responsable:** Equipo SISOC  
    **Nivel:** Interno

## Mobile first
Interfaz pensada para teléfono, botones grandes, lectura rápida.

## Simplicidad operativa
El centro debe poder operar sin asistencia técnica.

## Trazabilidad
Toda acción relevante se registra (quién / qué / cuándo / resultado).

## Alcance congelado
El MVP incluye 5 módulos. Lo demás va a Release 2.
